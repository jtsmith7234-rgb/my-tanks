### Issues Found & Fixed

| ID | Severity | Issue | Fix Applied |
|----|----------|-------|-------------|
| W-1 | Medium | `.input.tiny` font-size was 13px — triggers iOS keyboard zoom on focus (input fields like water parameter counts) | Fixed: `font-size:16px` in `styles.css` |
| W-2 | Medium | Info.plist had landscape orientations enabled — app is portrait-only, landscape exposes layout issues | Fixed: locked to `UIInterfaceOrientationPortrait` only in `Info.plist` |
| W-3 | Low | No `UIStatusBarStyle` key in Info.plist — native status bar appearance undefined in Capacitor shell | Fixed: added `UIStatusBarStyleDefault` (dark text, suits light themes) |
| W-4 | Low | No `format-detection` meta tag — iOS might auto-link numbers in tank parameter displays | Fixed: added `<meta name="format-detection" content="telephone=no, date=no, address=no, email=no">` |

## Exact Next Steps for Xcode / Device

### On your Mac:

```bash
# 1. Open Terminal, navigate to the iOS project
cd ~/path/to/tank-care-buddy/ios/App

# 2. Install CocoaPods dependencies (required before opening in Xcode)
pod install

# 3. Open the workspace in Xcode (use .xcworkspace, NOT .xcodeproj)
open App.xcworkspace
```

### In Xcode:

4. Select your development team: **Signing & Capabilities** tab → **Team** → select your Apple ID
5. Confirm **Bundle Identifier** is `com.tankcarebuddy.app`
6. Select a simulator (e.g., iPhone 15 Pro) or plug in your iPhone
7. Press **Run** (▶)

### After first run:
8. Test: open a tank, switch tabs, open modals, check bottom spacing on notched iPhone
9. Check status bar — if text is invisible, change `UIStatusBarStyle` to `UIStatusBarStyleLightContent` in Info.plist for dark themes
